# CS50’s Introduction to Databases with SQL

This repository contains my solutions, notes, and practice files for **CS50’s Introduction to Databases with SQL**, a course offered by Harvard University.

The course introduces relational databases and SQL, covering database design, querying, normalization, indexing, and transactions.

---

## Course Topics

Key topics covered in this repository include:

- Relational databases and data modeling
- SQL fundamentals (SELECT, INSERT, UPDATE, DELETE)
- Joins, subqueries, and views
- Normalization and schema design
- Indexes and performance
- Transactions and concurrency
- SQLite and PostgreSQL
